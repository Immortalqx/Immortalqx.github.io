Vision 1.0

这是我们第一代的物流无人机，这款无人机设计于2020年，最初被用于参加中国机器人大赛无人机挑战赛。。。。。。

这款无人机的设计思路是：xxxxxxxx。采用了xxx方法，通过xxxx解决了xxxx问题。

但是这款无人机还存在一些问题：

  • 定位的精度不够。

    SLAM初始化之后的尺度并不好，比如将5米计算成3米或者6米。这一定程度上影响了无人机投放的准度。

  • 只对投放靶进行了识别，没有计算靶心相对于无人机的位置。

    为了保证投放精确，我们通过保持靶心在下视图像中心来确保无人机能够朝靶心投放快递，这样虽然能够解决问题，但是控制难度过高（很难确定无人机应该往哪个方向运动多少米），也影响了无人机投放的准度。

  • 无人机设计的过于笨重，飞行控制非常困难。

    在检测到投放靶的时候，我们希望控制无人机下降一定高度来完成更精准的投放，但当无人机携带的快递重量过重时，无人机在下降高度时会远远超出预期，只能够通过减少快递的重量来保证无人机正常工作。

My works:

  • 为无人机安装了realsense D435i相机和realsense T265相机
  • 在无人机上部署了open-vins算法和vins-fusion算法，保证无人机能够通过它们获取自己的位姿
  • 通过构建docker来减少环境配置的工作量，同时加快算法在不同无人机上的部署速度

Cave Exploration UAV

TODO: describe this project and upload the video!!!

这是基于多传感器融合的无人机洞穴探测项目（国家级大创），我是这个项目的负责人。

本项目旨在通过多传感器融合技术实现一款能够在人工作业前进入洞穴进行初步勘探、有效地降低洞穴探测事故发生风险的无人机洞穴探测方案。该无人机将搭载多种传感器，包括高清摄像头、红外摄像头、激光雷达、GPS、IMU等，并且能够感知自身姿态、相对地面的高度，在全黑的条件下也能够通过红外摄像头、激光雷达等对外界进行有效探测，同时不需要人的干预，可以实时的对山洞进行三维建图并自主返回。

My main work is as follows:

  • 传感器的选择、标定
  • LIS系统的选择与部署
  • VIS系统的选择与部署
  • LIS系统与VIS系统的融合【正在进行中！】

Pi-SLAM-Fusion: UAV Image Mosaicing based on Pi-SLAM and Map2DFusion

TODO: describe this project and upload the video!!!

这是一个结合了Pi-SLAM和Map2DFusion的程序。

My main work is as follows:

  • 阅读了Pi-SLAM和Map2DFusion的核心代码，梳理了两个方案的思路

  • 调整Pi-SLAM和Map2DFusion的依赖项，让两个方案能够一起编译通过

  • 在Pi-SLAM点云计算的部分插入了RANSAC算法，用于拟合点云中的地平面（默认相机对地拍摄）

  • 将Pi-SLAM和Map2DFusion合并。

    Map2DFusion作为Pi-SLAM的一个线程，Pi-SLAM启动之后计算一定帧数的位姿和点云，通过RANSAC算法拟合点云中的地平面，拟合完毕之后发送给Map2DFusion线程来启动该线程，之后Pi-SLAM将计算出位姿的每一帧发送给Map2DFusion，由Map2DFusion进行实时的2D地图拼接。

NWPU Soccer Robot Base SLAM Group Learning Tutorial

This project was initiated by the NWPU Soccer Robot Base SLAM Group, with Quanxiang Liu as the main leader.

We made this tutorial for better and more standardized recruiting training at the base, and of course we hope to help students who are interested in SLAM technology. Through the tutorial, we hope to give the team members an overall knowledge of SLAM, and to quickly master the basic positioning techniques so that they can be used in the base’s competitions or projects.

PS: This project is still in the development stage and there are some tutorials that are not yet completed.
