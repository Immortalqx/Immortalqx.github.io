航拍笔记（零）：准备、检查与保养

一、前期准备

  1. 查询拍摄城市的无人机禁（限）飞区，这一点可以去无人机厂家官网、当地政府部门网站等地方查询；
  2. 提前观察飞行环境，做好航线规划，选择四周及上方空旷、地面平整的场地作为起降点；
  3. 关注天气预报，确认拍摄当天的天气状况，避免在大风、雨雾、雷电等恶劣天气飞行；
  4. 对无人机所有部件进行详细检查，确保无损伤，特别要注意检查动力电池的健康状态。

二、起飞检查

  1. 确认自己已获得飞行许可，或不在禁（限）飞区；
  2. 如果环境温度低于15摄氏度，建议将动力电池预热（可用车载空调）到30摄氏度以上，极寒天气还应对电池做好保温措施；
  3. 设置安全的自动返航高度，同时建议记下一个起降点附近的明显参照物，以备出现紧急情况时快速找到返航点；
  4. 再次检查无人机各部件是否有损伤、连接是否正常、可靠，动力电池、地面站（遥控、图传）电量充足；
  5. 起飞前对无人机做一次指南针校准，并确认无人机已记录返航点；
  6. 飞手精神状态良好，如果近视请带上眼镜；
  7. 起飞前，注意引导围观者（特别是好奇心重的小朋友）远离无人机；

三、飞行期间的注意事项

  1. 飞行切记炫耀、浮躁，飞手应该保持沉稳冷静的心态；
  2. 避免无人机在人群密集处上空悬停，远离电力线、通信线、移动通信塔等可能的干扰源；
  3. 尽可能保持无人机在视距内飞行，最好不要绕飞至建筑物背后（远离起飞点的一侧），因为这样容易导致无人机失去遥控（图传）信号；
  4. 航拍时，云台相机视角常常与第一飞行视角不一致，飞手应留意无人机机头朝向，避免出现误操作；
  5. 关注监视器上无人机航行仪表信息，如飞行途中出现异常，应立即查明原因并排除，必要时中止飞行，并立即返航降落；
  6. 随时关注无人机动力电池电量，切忌低电量贪飞，确保有足够的时间返航。

四、返航降落

  1. 拍摄结束或电量不足返航时，注意避开障碍物，保持足够的净空安全距离；
  2. 对于旋翼飞行器，下降（竖直）速度不宜过快，过快的下降速度容易导致无人机姿态失稳，可采用斜向返航的飞行方式增强无人机姿态稳定性；
  3. 着陆前瞬间，俯仰、横滚舵宜归中，无人机不要出现大幅度水平移动，避免侧翻；
  4. 着陆后，无人机螺旋桨完全停止转动前，所有人员不要靠近无人机；
  5. 航拍作业完成，先关无人机动力电源，再关地面站（遥控、图传）电源；
  6. 飞行结束，请参照原厂说明书对电池进行保养。

五、保养

电池

  1. 避免在强静电或者磁场环境中使用电池。

  2. 请勿在电池电源打开的状态下插拔电池，若电池接口有污物，可用干布擦拭干净。

  3. 电池应在 -10℃ 至 40℃* 的环境下使用，当飞行环境低于 5℃ 时，应提前将电池放置在常温环境下进行预热，充分预热至 20°C 以上。

    部分机型如 Mavic Mini 的工作环境为 0℃ 至 40℃；

    悟 Inspire 2 可在 -20℃ 至 40℃ 环境下工作 ，且有自加热功能。

  4. 如有破损、膨胀、漏液等异常现象，请勿再次使用。

    电池内部液体有强腐蚀性，如有泄露，请远离。

  5. 电池可充电环境温度为 5℃ - 40℃，理想的充电环境温度（22℃-28℃）可大幅度延长电池寿命，飞行结束后，建议待降至室温后再进行充电。

电池存放建议：

  • 设备不使用时，电池需单独存放；
  • 保持存放环境干燥，不可将电池放置于靠近热源、易燃易爆品的区域；
  • 避免电池长期放置在低温的室外，否则电池活性将大大降低，甚至造成锂电池性能不可逆的下降；
  • 电池请勿在完全放电状态下长期放置，以免电池进入过放状态造成电芯损害，应将电池放电至 40%-65% 电量进行存放；
  • 建议 2-3 个月重新充放电一次，以保证电池活性；
  • 长期存储时，在-10°C~45°C范围内的环境中存放。

电机

检查电机无沙尘和水渍，并活动各个电机轴，确认与机臂固定牢固且能顺畅达到最大位置，然后卸下桨叶后开机并启动电机，判断电机有无异响。

桨叶

检查桨叶是否有破损、变形，若存在异常，则更换新的桨叶。在飞行时，建议给桨叶安装桨叶保护罩。

云台相机

检查镜头无划痕、破损、污垢，云台卡扣是否有异物、排线是否正常连接。必要时，对云台相机做一个自动校准，点选 App 中的云台设置——云台自动校准。

视觉系统

检查视觉避障系统与红外传感器镜片是否有划痕、破损或污渍。

遥控器

检查遥控器天线是否有损坏，以及按键是否可以正常按压反馈，并且遥控器遥杆等是否正常。

App和固件版本

及时更新 app 和固件版本，提升拍摄体验的同时能获得得到更好的拍摄效果。
