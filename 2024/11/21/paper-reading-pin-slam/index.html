и®әж–Үйҳ…иҜ»гҖҠPIN-SLAM: LiDAR SLAM Using a Point-Based Implicit Neural Representation for Achieving Global Map ConsistencyгҖӢ

и®әж–ҮжғіиҰҒи§ЈеҶізҡ„й—®йўҳпјҡThese SLAM approaches lack support for direct loop closure corrections and are not able to build globally consistent maps of larger scenes. This is mainly due to the usage of regular grid-based local feature embeddings, which are not elastic and resilient to loop corrections.

и®әж–Үзҡ„ж ёеҝғзӣ®ж ҮпјҡThe main focus of this work is a LiDAR SLAM system for building globally consistent maps using a PIN map representation.

и®әж–Үдёәд»Җд№ҲйҖүжӢ©neural pointиЎЁзӨәеңәжҷҜпјҡWe opt to use a neural point-based implicit representation, which has two main advantages over grid-based representations: the flexibility of spatial distribution and the elasticity for transformations. Recent work (Point-SLAM) only makes use of the first advantage for neural RGB-D SLAM at the cost of scalability and inefficient neighborhood querying. Instead, we exploit the second and more important advantage to build a globally consistentmap that can be corrected while online mapping after closing a loop, which is essential for large-scale LiDAR-based SLAM.

и®әж–ҮеңЁIntroductionжҸҗеҲ°пјҢMappingйғЁеҲҶдё»иҰҒжҳҜеҹәдәҺд№ӢеүҚзҡ„е·ҘдҪңSHINE-MappingпјҢиҖҢOdometryйғЁеҲҶжҳҜжү©еұ•дәҶд№ӢеүҚзҡ„е·ҘдҪңLocNDFгҖӮ

е·ҘдҪңжҳҜе»¶з»ӯзҡ„пјҢйқһеёёжүҺе®һпјҢд»ӨдәәзҫЎж…•гҖӮ

и®әж–ҮжҸҗеҮәдәҶеӣӣдёӘkey claimsпјҡ

  1. Our SLAM system achieves localization accuracy better or on par with state-of-the-art LiDAR odometry/SLAM approaches and is more accurate than recent implicit neural SLAM methods on various datasets using different range sensors.
  2. Our method can conduct large-scale globally consistent mapping with loop closure thanks to the elastic neural point representation.
  3. Our map representation is more compact than the previous counterparts and can be used to reconstruct accurate and complete meshes at an arbitrary resolution.
  4. Our correspondence-free scan-to-implicit map registration and the efficient neural point indexing by voxel hashing enable our algorithm to run at the sensor frame rate on a single NVIDIA A4000 graphics processing unit (GPU).

и®әж–ҮеҜ№еҹәдәҺSubmapзҡ„ж–№жі•зҡ„иҜ„д»·пјҡThe submap-based methods accumulate the observations as a submap and assume the submap is a locally defined rigid body. The pseudoglobal consistency is maintained by optimizing a graph linking submaps and their associated poses through submap-to-submap registrations. The usage of submaps decreases the number of nodes and edges in PGO, thus saving computational effort. However, these methods have issues of ambiguity in the overlapping region of the adjacent submaps and determining appropriate criteria for submap division to balance the rigidity and efficiency.

  2. OdometryпјҡйҖҡиҝҮе°ҶзӮ№дә‘$P_r$дёҺlocal map $M_l$зҡ„implicit SDFиҝӣиЎҢй…ҚеҮҶжқҘдј°и®Ўе…ЁеұҖзҡ„дҪҚе§ҝ$T_{WC_t}$пјҢе№¶е°ҶеүҚеҗҺдёӨеё§зҡ„еҸҳжҚў$T_{C_{t-1}C_t}$дҪңдёәдёҖжқЎиҫ№ж·»еҠ еҲ°Pose Graph $G$дёӯгҖӮ

  5. Pose graph optimizationпјҡеҠ е…Ҙloop closure edgeд№ӢеҗҺпјҢеҜ№pose graph $G$иҝӣиЎҢдјҳеҢ–гҖӮдјҳеҢ–д№ӢеҗҺеҜ№global map $M$дёӯжҜҸдёӘneural pointзҡ„дҪҚзҪ®е’Ңж–№еҗ‘еҸҠе…¶е…іиҒ”её§иҝӣиЎҢеҸҳжҚўпјҢд»ҘдҝқиҜҒе…ЁеұҖдёҖиҮҙжҖ§гҖӮеңЁе®ҢжҲҗloop correctionд№ӢеҗҺпјҢеҜ№и®ӯз»ғж ·жң¬жұ $D_p$иҝӣиЎҢеҸҳжҚўпјҢе№¶дё”йҮҚзҪ®local map $M_l$гҖӮ

Neural Point-Based Map Representation

Map Representationпјҡ

neural point $m_i$дҝқеӯҳдәҶиҮӘе·ұзҡ„дҪҚзҪ®$x_i$пјҢе№¶дё”жңүдёҖдёӘеӣӣе…ғж•°$q_i$иЎЁзӨәиҮӘиә«еқҗж ҮжЎҶжһ¶зҡ„ж–№еҗ‘пјҢзү№еҫҒеҗ‘йҮҸ$f^g_i$зј–з ҒдәҶеұҖйғЁзҡ„еҮ дҪ•еҪўзҠ¶пјҢж—¶й—ҙжҲі$t^c_i$иЎЁзӨәиҝҷдёӘneural pointеҲӣе»әзҡ„ж—¶й—ҙпјҢж—¶й—ҙжҲі$t^u_i$иЎЁзӨәиҝҷдёӘneural pointжңҖеҗҺжӣҙж–°зҡ„ж—¶й—ҙпјҢ$\mu_i$иЎЁзӨәиҜҘзӮ№жҳҜеҗҰжҙ»и·ғгҖҒжҳҜеҗҰзЁіе®ҡгҖӮйҖҡиҝҮе°Ҷneural point $m_i$дёҺ$t^m_i$ж—¶еҲ»зҡ„дҪҚе§ҝ$T_{WC_{t^m_i}}$зӣёе…іиҒ”пјҢеҸҜд»ҘйҖҡиҝҮжӣҙж–°жүҖжңүж—¶еҲ»зҡ„дҪҚе§ҝжқҘзӣҙжҺҘж“Қзәөең°еӣҫпјҢе…¶дёӯ$t^m_i = [(t^c_i + t^u_i)/2]$гҖӮ

е…¶дёӯиҝҷдёӘеӣӣе…ғж•°$q_i$пјҢжҳҜз”ЁжқҘдҝқиҜҒneural pointз»ҸиҝҮdecoderи§Јз Ғеҫ—еҲ°зҡ„SDFеҜ№еҲҡдҪ“еҸҳжҚўе…·жңүдёҚеҸҳжҖ§гҖӮеҰӮдёӢеӣҫ(a)йғЁеҲҶжүҖзӨәпјҢdecoder$D^g_\theta$иҫ“е…Ҙзҡ„жҳҜзӣёйӮ»neural pointзҡ„feature $f^g_i$е’ҢеңЁиҜҘneural pointеқҗж Үзі»дёӢиў«жҹҘиҜўзӮ№зҡ„дҪҚзҪ®$d_i$пјҢиҫ“еҮәзҡ„жҳҜиў«жҹҘиҜўзӮ№еңЁиҜҘneural pointеҪұе“ҚдёӢзҡ„SDFеҖј$s_i$гҖӮж•ҙдёӘиҝҮзЁӢдёҺPoint-NeRFе’ҢPoint-SLAMи®Ўз®—дҪ“еҜҶеәҰзҡ„иҝҮзЁӢеҹәжң¬дёҖиҮҙгҖӮд»ҺдёӢеӣҫ(b)йғЁеҲҶеҸҜд»ҘзңӢеҲ°пјҢиҝҷз§Қзӯ–з•ҘдҪҝеҫ—йў„жөӢзҡ„SDFеҖјеҜ№еұҖйғЁзҡ„е№із§»е’Ңж—ӢиҪ¬е…·жңүдёҚеҸҳжҖ§пјҢд»ҺиҖҢдҪҝеҫ—з»ҸиҝҮloop closureд№ӢеҗҺзҡ„ең°еӣҫе…·жңүеј№жҖ§гҖӮ

з¬”иҖ…и®ӨдёәиҝҷдёҖжӯҘеҒҡзҡ„йқһеёёе·§еҰҷгҖӮеӣӣе…ғж•°$q_i$е®ҡд№үдәҶжҜҸдёҖдёӘneural pointиҮӘе·ұзҡ„еқҗж Үзі»пјҢжҹҘиҜўSDFйңҖиҰҒжҠҠneural pointзҡ„featureе’ҢжҹҘиҜўзӮ№еңЁеқҗж Үзі»дёӯзҡ„дҪҚзҪ®иҫ“е…ҘеҲ°зҪ‘з»ңпјҢзӣёеҪ“дәҺжҳҜжҜҸдёҖдёӘneural pointйғҪжңүдәҶиҮӘе·ұзҡ„вҖңеҪўзҠ¶вҖқпјҢиҝҷдәӣвҖңеҪўзҠ¶вҖқеҸ еҲ°дёҖиө·жһ„жҲҗдәҶдёҖдёӘеұҖйғЁзҡ„еҮ дҪ•иЎЁйқўгҖӮз»ҸиҝҮloop closureд№ӢеҗҺжҜҸдёӘвҖңеҪўзҠ¶вҖқд№ҹдјҡзӣёеә”зҡ„иў«ж—ӢиҪ¬е№із§»пјҢиҖҢдёҚжҳҜз®ҖеҚ•зҡ„жҠҠвҖңеҪўзҠ¶вҖқзҡ„дёӯеҝғжҢӘеҲ°ж ЎжӯЈд№ӢеҗҺзҡ„дҪҚзҪ®еҺ»пјҢиҝҷж ·зҡ„иҜқе°ұдёҚйңҖиҰҒйҖҡиҝҮеӯҗең°еӣҫжҠҠжҜҸдёӘзӮ№зҡ„зӣёеҜ№дҪҚзҪ®з»‘иө·жқҘд№ҹиғҪдҝқиҜҒloop closureд№ӢеҗҺеұҖйғЁзҡ„еҮ дҪ•зЁіе®ҡдәҶгҖӮ

иҝҷдёӘиҝҮзЁӢжңүдёҖзӮ№3DGSзҡ„е‘ійҒ“дәҶпјҢдҪҶжҳҜеҚ•дёӘ3DGSеҫҲйҡҫеғҸиҝҷйҮҢзҡ„еҚ•дёӘneural pointйӮЈж ·жңүйқһеёёжҳҺзЎ®зҡ„еҮ дҪ•ж„Ҹд№үгҖӮ

з–‘й—®пјҡдёәд»Җд№ҲPoint-NeRFгҖҒPoint-SLAMгҖҒLoopy-SLAMд»ҘеҸҠPIN-SLAMеҜ№дәҺдҪ“еҜҶеәҰжҲ–SDFеҖјпјҢйғҪжҳҜеҚ•зӢ¬и§Јз Ғд№ӢеҗҺпјҢеҶҚйҖҡиҝҮйҖҶи·қзҰ»еҠ жқғпјҹиҖҢйўңиүІзӣёе…ізҡ„пјҢеҲҷжҳҜйҖҡиҝҮзҪ‘з»ңеҫ—еҲ°дёҖдёӘж–°зҡ„featureпјҢе°ҶиҝҷдёӘfeatureиҝӣиЎҢйҖҶи·қзҰ»еҠ жқғд№ӢеҗҺеҶҚи§Јз Ғеҫ—еҲ°йўңиүІпјҹ

Map Data Structureпјҡ

дёәдәҶе®һзҺ°еҝ«йҖҹзҡ„neural pointзҙўеј•е’ҢйӮ»еҹҹжҗңзҙўпјҢPIN-SLAMз»ҙжҠӨдәҶдёҖдёӘvoxel resolutionеӣәе®ҡдёә$v_p$е’Ңhash table sizeдёә$T$зҡ„voxel hashing data structureгҖӮиҝҷдёӘж•°жҚ®з»“жһ„з”ЁдәҺз»„з»Үең°еӣҫдёӯзҡ„neural pointпјҢдҝқиҜҒжҜҸдёӘvoxelдёӯзҡ„neural pointдёҚи¶…иҝҮ1дёӘгҖӮ

PIN-SLAMеҖҹйүҙдәҶгҖҠInstant neural graphics primitives with a multiresolution hash encodingгҖӢе’ҢгҖҠвҖңKISS-ICP: In defense ofpoint-to-point ICPвҖ“simple, accurate, and robust registration if done the right wayгҖӢиҝҷдёӨдёӘе·ҘдҪңпјҢйҖҡиҝҮspatial hashing function $\kappa = h(p_W)$е°Ҷ$p_W \in \mathbb{R}^3$жҳ е°„еҲ°hash tableдёӯзҡ„$\kappa \in \mathbb{Z}$гҖӮе…¶дёӯhash tableдёӯжҜҸдёҖйЎ№еӯҳеӮЁең°еӣҫдёӯneural pointзҡ„зҙўеј•пјҢй»ҳи®ӨеҖјдёә-1пјҢиЎЁзӨәиҝҷдёҖйЎ№жҡӮж—¶жІЎжңүеҜ№еә”зҡ„neural pointгҖӮ

дёҺPoint-SLAMзӣёжҜ”пјҢPIN-SLAMеҖҹеҠ©voxel structureпјҢе®һзҺ°дәҶжӣҙеҝ«зҡ„иҝӣиЎҢKиҝ‘йӮ»жҗңзҙўгҖӮ

Map Initialization and Updateпјҡ

PIN-SLAMдјҡж №жҚ®жҜҸдёӘж—¶еҲ»иҫ“е…Ҙзҡ„зӮ№дә‘еҲқе§ӢеҢ–жҲ–жӣҙж–°neural pointгҖӮ

еҜ№дәҺеңЁж—¶еҲ»$t$иҺ·еҸ–зҡ„дё–з•Ңеқҗж Үзі»$W$дёӢзҡ„жҹҗдёҖдёӘзӮ№$p_W$пјҢж»Ўи¶ід»ҘдёӢд»»ж„ҸдёҖз§Қжғ…еҶөж—¶пјҢдјҡеҲқе§ӢеҢ–дёҖдёӘж–°зҡ„neural point $m_k$ж”ҫеҲ°ең°еӣҫ$M$дёӯпјҢе№¶дё”е°ҶеҜ№еә”зҡ„hash tableдёӯзҡ„йЎ№и®ҫзҪ®дёәиҜҘneural pointзҡ„зҙўеј•пјҢеҚі$\kappa = k$гҖӮ

  1. еҜ№еә”зҡ„hash tableдёӯзҡ„йЎ№жІЎжңүиў«еҚ з”ЁпјҢеҚі$\kappa = -1$пјӣ
  2. еӯҳеңЁе“ҲеёҢеҶІзӘҒпјҢдҪҶжҳҜеңЁvoxel structureдёӯпјҢеӯҳеӮЁзҡ„neural pointдёҺеёҢжңӣиў«ж·»еҠ зҡ„neural pointд№Ӣй—ҙи·қзҰ»еҫҲиҝңпјӣ
  3. еӯҳеңЁе“ҲеёҢеҶІзӘҒпјҢдҪҶжҳҜеӯҳеӮЁзҡ„neural pointе·Із»ҸдёҚеҶҚжҙ»и·ғпјҢеҚі$D(t) - D(t^u_\kappa) > d_l$гҖӮ

ж–°еҲқе§ӢеҢ–зҡ„neural point $m_k$пјҢдҪҚзҪ®$x = p_W$пјҢиҖҢеӣӣе…ғж•°иў«и®ҫзҪ®дёәеҚ•дҪҚеӣӣе…ғж•°$ q = (1,0,0,0)$пјҢзү№еҫҒеҗ‘йҮҸиў«еҲқе§ӢеҢ–дёәйӣ¶еҗ‘йҮҸ$f^g=0$пјҢеҲӣе»әж—¶й—ҙе’ҢжңҖеҗҺжӣҙж–°ж—¶й—ҙиў«и®ҫзҪ®дёә$t^c = t^u = t$пјҢзЁіе®ҡжҖ§$\mu = 0$гҖӮ

Map Training Samplesпјҡ

е’ҢPoint-SLAMзҡ„йҮҮж ·зӯ–з•ҘжҜ”иҫғзұ»дјјпјҢPIN-SLAMжҳҜеңЁиЎЁйқўйҮҮж ·дёҖдёӘзӮ№пјҢеҶҚеңЁиЎЁйқўйҷ„иҝ‘йҮҮж ·$Ns$дёӘзӮ№пјҢиЎЁйқўйҷ„иҝ‘зҡ„$N_s$дёӘзӮ№зҡ„ж·ұеәҰжңҚд»ҺжӯЈжҖҒеҲҶеёғгҖӮеҸҰеӨ–пјҢд№ҹдјҡеңЁиЎЁйқўеҲ°зӣёжңәд№Ӣй—ҙзҡ„з©әй—ҙдёӯеқҮеҢҖйҮҮж ·$N_f$дёӘзӮ№пјҢеңЁиЎЁйқўеҗҺзҡ„дёҖе®ҡз©әй—ҙдёӯеқҮеҢҖйҮҮж ·$N_b$дёӘзӮ№гҖӮ

жңҖз»Ҳдјҡеҫ—еҲ°еҰӮдёӢжүҖзӨәзҡ„и®ӯз»ғж ·жң¬пјҡ
$$
D = {(u_j,\hat s_j , t) | j = 1,вҖҰ,N_t}
$$
е…¶дёӯпјҢ$N_t = M_t(1+N_s+N_f+N_b)$пјҢ$M_t$жҳҜз”ЁдәҺmappingзҡ„зӮ№дә‘$P_m$дёӯзӮ№зҡ„дёӘж•°пјҢ$u_j$иЎЁзӨәз¬¬$j$дёӘйҮҮж ·зӮ№пјҢ$\hat s_j$иЎЁзӨәSDFеҖјпјҢиў«е®ҡд№үдёә$\hat s_j = |p|_2 - d$гҖӮи®әж–ҮжҸҗеҲ°$\hat s_j$еҸҜиғҪжҖ»жҳҜдјҡиў«й«ҳдј°пјҢдҪҶжҳҜи®Ўз®—йҖҹеәҰеҫҲеҝ«пјҢиҖҢдё”еңЁж ·жң¬жҺҘиҝ‘иЎЁйқўзҡ„ж—¶еҖҷжҳҜдёҖдёӘеҫҲеҘҪзҡ„иҝ‘дјјеҖјгҖӮ

дёәдәҶи§ЈеҶіincremental mappingдёӯзҡ„вҖңcatastrophic forgettingвҖқпјҢPIN-SLAMз»ҙжҠӨдәҶдёҖдёӘи®ӯз»ғж ·жң¬жұ $D_p$пјҢзұ»дјјдәҺPoint-SLAMзӯүж–№жі•дёӯзҡ„е…ій”®её§гҖӮ

Loop Closure Detection

PIN-SLAMйҰ–е…ҲжЈҖжөӢlocal loop closureгҖӮеҚіпјҢеҜ№дәҺеҪ“еүҚдҪҚзҪ®$t_t$е’ҢеҪ“еүҚеё§$C_t$пјҢдёҺеҺҶеҸІдҪҚзҪ®$t_{t_i}$е’ҢеҺҶеҸІеё§$C_{t_i}$пјҢе…¶дёӯ$t_i < t$иҖҢдё”$D(t) - D(t_i) > d_l$пјҢеҲӨж–ӯдёӨдёӘдҪҚзҪ®жҳҜеҗҰж»Ўи¶ід»ҘдёӢе…ізі»пјҡ
$$
| t_t - t_{t_i} |2 < d{loop}
$$
иҝҷз§Қlocal loop closureжҜ”иҫғйҖӮеҗҲеңЁdriftиҫғе°Ҹзҡ„жғ…еҶөдёӢпјҢеҲӨж–ӯжҳҜеҗҰеҸҲеӣһеҲ°дәҶиҝҮеҺ»з»ҸиҝҮзҡ„дҪҚзҪ®гҖӮ

еҰӮжһңжІЎжңүжЈҖжөӢеҲ°local loop closureпјҢPIN-SLAMдјҡйҖҡиҝҮжҜ”иҫғframe-wise descriptorsжқҘжҗңзҙўglobal loop closureгҖӮPIN-SLAMй’ҲеҜ№е…ёеһӢзҡ„й©ҫй©¶еңәжҷҜпјҢеҸӮиҖғScan Contextе’ҢScan Context++зӯүе·ҘдҪңпјҢжҸҗеҮәдәҶдёҖз§ҚвҖңneural point feature enhanced local map descriptorвҖқпјҢи®әж–ҮдёәжҜҸдёҖдёӘж—¶еҲ»$t$зҡ„local mapз”ҹжҲҗеҜ№еә”зҡ„local context descriptor $U_t$гҖӮ

еҪ“жЈҖжөӢеҲ°candidate loopпјҢPIN-SLAMе°ҶеҪ“еүҚеё§зӮ№дә‘$P_r$дёҺеҖҷйҖүеё§$C_k$еҜ№еә”зҡ„local map $M_l$иҝӣиЎҢй…ҚеҮҶгҖӮ

  • еҜ№дәҺglobal loopпјҢдҪҝз”Ёsemimetric localizationзҡ„з»“жһңдҪңдёәеҲқеҖјгҖӮ
